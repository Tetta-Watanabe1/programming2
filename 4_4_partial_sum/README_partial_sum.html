プログラミングII演習課題の手引き
4_4_partial_sum

1 説明

整数配列の部分和を出力したい．引数で与えられた配列pの先頭w個の要素の和を求める関数int partial_sum(int *p, int w)関数を作成せよ．

また，main関数ではpartial_sum関数を呼び出して，この関数の戻り値をsumに代入せよ．

partial_sum関数の引数：

  • a[s]のアドレスpと整数wの二つを引数にする．a[s]のアドレスとは配列aのs番目の要素のアドレス．

partial_sum関数の戻り値： p[0] + p[1] + p[2] + ... + p[w-1]

pがmain関数の配列aのs番目の要素のアドレスであれば，上の和は次のように配列aで表すことができる．

a[s] + a[s+1] + a[s+2] + ... + a[s+w-1]

入力：標準入力に2つの整数sとwが与えられる．

出力：配列aの要素のうちs番目からw個の要素の和を出力する．つまり次式の和を計算して出力する．なお，配列aはmain関数にあらかじめ定義されている．

    a[s] + a[s+1] + a[s+2] + ... + a[s+w-1]

コンパイル：

    $ gcc -Wall -o partial_sum partial_sum.c

4 ヒント

ヒント：

  • 明快入門C 67 ポインタによる配列渡し，66 ポインタを引数にする．

  • 次のようにpartial_sum関数を作って欲しい．難しい人は以下の代案1から 3のいずれかでもよい．

  • main関数からpartial_sum(&a[s], w);のように呼び出すと，配列aの s番目の要素のアドレスが第1引数として渡される．

  • int partial_sum(int *pa, int w)と宣言する．

  • 仮引数paで部分和を求める先頭要素へのポインタ（main関数の&a[s]）を受け取り，仮引数wで部分和を求める要素数を受け取る．このとき， pa[0]からpa[w - 1]がmain関数のa[s]からa[s + w - 1]に対応する．

  • pa[0]からpa[w - 1]の和を返す．この和は次式のsumで表せる．

    \[{\tt sum} = \sum_{i=0}^{\tt w-1} {\tt pa}[i]\]

  • sumは，先頭要素から順に要素の和を求め，ポインタを次の要素に移動することをw回繰り返すことで求められる．

    int sum;
sum = 0;
while (w-- > 0) {
  sum += *pa;
  pa++;
}

    同じ計算を配列の添え字iを使って書くと次のようになる．

    int sum, i;
sum = 0;
i = 0;
while (i < w) {
  sum += pa[i];
  i++;
}

  • while文の代わりにfor文でも書ける．

    int sum;
for (sum = 0; w-- > 0; pa++) {
  sum += *pa;
}

  • sum += *pa;とpa++;の二文は次のように一文にまとめることもできる．

    sum += *pa++;

    *pa++の二つの演算子*と++は同じ優先順位で右結合なので *(pa++)の順に評価される．(++pa)ではないので，pa++で参照される変数は，ポインタpaを増やす前のアドレスに格納されている変数であることに注意せよ．

  • （代案1）配列aの先頭アドレスを関数に渡し，関数内ではポインタを使う．

    • main関数からpartial_sum(a, s, w);のように呼び出す．

    • int partial_sum(int *pa, int s, int w)と宣言する．

    • 仮引数paで配列a（の先頭要素）へのポインタを受け取る．

    • 仮引数sとwでそれぞれ先頭要素の添字と要素数を受け取る．

    • pa[s]からpa[s + w - 1]の和を返す．

    • pa += s;でポインタpaをs番目の要素に移動する．

  • （代案2）関数本体ではポインタを使わない書き方．

    • main関数からpartial_sum(&a[s], w);のように呼び出す．

    • int partial_sum(int b[], int w)と宣言する．

    • b[0]からb[w - 1]の和sumを返す．

      \[{\tt sum} = \sum_{i=0}^{\tt w-1} {\tt b}[i]\]

  • （代案3）これが一番理解しやすいはず．ポインタをまったく使わないのでポインタの課題としては不十分．

    • main関数からpartial_sum(a, s, w);のように呼び出す．

    • int partial_sum(int b[], int s, int w)と宣言する．

    • 次式のsumを返す．

      \[{\tt sum} = \sum_{i={\tt s}}^{\tt s+w-1} {\tt b}[i]\]
