2_7_sudoku_print

1 説明

発展課題

配列int grid[9][9]に格納された数独の盤面を出力する sudoku_print(grid[9][9]) 関数を作成せよ．

数独のルール：

  • タテ9列，ヨコ9行，3 \(\times\) 3の9つあるブロックのどれにも，1から9までの数字が一つずつ入るように空きマスに数字を入れる．

  • 詳しくは以下のページで確認してください．なお「数独(SUDOKU)」はニコリの登録商標です．

    http://nikoli.co.jp/ja/puzzles/sudoku/ (和文)

    http://nikoli.co.jp/en/puzzles/sudoku/ (英文)

  • 参考：数独の空きマスの最大数に関する論文．

    Gary McGuire, Bastian Tugemann, and Gilles Civario, “There is no 16-Clue Sudoku: Solving the Sudoku Minimum Number of Clues Problem via Hitting Set Enumeration,” Journal of Experimental Mathematics, vol. 23, no. 2, Taylor & Franscis, Aug. 2013. https://doi.org/10.1080/10586458.2013.870056 https://arxiv.org/pdf/1201.0749.pdf

入力：なし（main関数の配列grid）

出力：

  • テスト一覧に示すように3 \(\times\) 3ブロックの縁を +-| で囲んで，13行で表す．

  • 配列要素が0のマスは空白にする．

コンパイル：

    $ gcc -Wall -o sudoku_print sudoku_print.c

4 ヒント

ヒント：

  • 明快入門C 19 配列（多次元配列），56 関数への引数渡し2 配列．

  • sudoku_print関数は2次元配列を引数に取るので，次のように宣言する．

    void sudoku_print(int grid[9][9]) {
}

  • sudoku_print関数の戻り値は不要なので，void型でよいだろう．

  • 2次元配列を引数にもつ関数を宣言する場合，次のどちらかのように配列のサイズを仮引数に書く必要がある．

       sudoku_print(int grid[9][9])
   sudoku_print(int grid[][9])

    なぜなら，配列の要素はメモリ上に[0][0]，[0][1]，[0][2]， \(\ldots\)，[0][8]，[1][0]，\(\ldots\)，[1][8]，[2][0], \(\ldots\)， [8][8]の順に格納されている．つまり，要素[i][j]はi*9+j番目に格納されている（先頭要素を0番目と数えていることに注意）．そのため，2次元目の要素数を与えないと，[i][j]が先頭から何番目に格納されているかわからない．一般に2次元以上の配列を引数にとる場合，最初の次元（添字）のみ省略可能であり，他はすべて指定しなければならない．
