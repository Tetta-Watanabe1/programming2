プログラミングII演習課題の手引き
3_6_sudoku_scanf

1 説明

発展課題

標準入力から数独の盤面を読み取り，盤面を出力せよ．

数独のルール：

  • タテ9列，ヨコ9行，3 \(\times\) 3の9つあるブロックのどれにも，1から9までの数字が一つずつ入るように空きマスに数字を入れる．

  • 詳しくは以下のページで確認してください．なお「数独(SUDOKU)」はニコリの登録商標です．

    http://nikoli.co.jp/ja/puzzles/sudoku/ (和文)

    http://nikoli.co.jp/en/puzzles/sudoku/ (英文)

  • 参考：数独の空きマスの最大数に関する論文．

    Gary McGuire, Bastian Tugemann, and Gilles Civario, “There is no 16-Clue Sudoku: Solving the Sudoku Minimum Number of Clues Problem via Hitting Set Enumeration,” Journal of Experimental Mathematics, vol. 23, no. 2, Taylor & Franscis, Aug. 2013. https://doi.org/10.1080/10586458.2013.870056 https://arxiv.org/pdf/1201.0749.pdf

コマンドライン引数：なし

入力：

  • 9 \(\times\) 9の盤面のそれぞれのマスを1文字で表す．

  • 盤面の1行を空白のない数字9つで表す．行の間には一つ以上の空白や改行が入る．実行例の場合は盤面3行おきに改行が入っている．

  • 引数の数字が0なら空きマス，1から9までの数字ならそのマスにその数字が入る．

出力：

  • テスト一覧に示すように3 \(\times\) 3ブロックの縁を+-|で囲んで，13行で表す．

  • 配列要素が0のマスは空白にする．

コンパイル：

    $ gcc -Wall -o sudoku_scanf sudoku_scanf.c

難しい人へ：

この手引きの末尾のヒントに沿って作ってください．

4 ヒント

  • 入力にscanf("%9s", ...)関数を使うことを想定して課題を作成している．とはいえ，getchar関数やscanf("%d", ...)関数を使って入力してもよい．

  • 入力にchar str[9]; scanf("%9s", str)関数を使う場合，変数strには数字が並んでいる．例えば"023456789"なら先頭は文字'0'であり，整数0ではない．次のようにstr[0]から0を引けば整数0になる．

    int n;
n = str[0] - '0';

  • 出力する部分は以前の課題で作成したsudoku_print関数を呼び出すようにするとよい．

  • 文字コードについてもう一度説明する．文字'3'と整数3はそのまま比較できない．次の例では”等しくない”と出力される．文字'3'や整数3を変数に代入した場合も出力は変わらない．

    if ('3' == 3)
  printf("等しい\n");
else
  printf("等しくない\n");

    文字定数’0’を文字から引けば整数値になる．次の例では”等しい”と出力される．

    if ('3' - '0' == 3)
  printf("等しい\n");
else
  printf("等しくない\n");
