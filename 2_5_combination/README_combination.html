2_5_combination

1 説明

階乗\(n!\)（1から\(n\)までの数の積）を求めるint factorial(int n)関数を作り，この関数を使って組合せの数 \(_n\mathrm{C}_r = \frac{n!}{r!(n-r)!}\) を出力するプログラムを作成せよ．

入力：

  • 標準入力に2つの整数\(n\)と\(r\)が，この順に与えられる．

  • \(n\)と\(r\)はいずれも整数で，\(1\le n < 12\)，\(1\le r \le n\)とする．

出力：

組合せの数\(_n\mathrm{C}_r\)を出力する．

コンパイル：

    $ gcc -o combination combination.c

#include <stdio.h>

/* ここに int factorial(int n) 関数を定義する */

int main() {
  int n, r;

  scanf("%d", &n);
  scanf("%d", &r);
  printf("%d\n", factorial(n) / factorial(r) / factorial(n - r));
  return 0;
}

4 ヒント

  • （数学） 階乗 \(n! = 1\cdot 2\cdot 3\cdot\ldots\cdot n\)

  • int factorial(int n)関数内での階乗の計算はforループと*=演算子を使うとよい．pow関数は整数ではなく，実数で計算するため，ここではおすすめしない．

余談：

  • 「2進数表現」の先取り： int型やlong型は整数を表すが，それぞれ表せる数値の最大値が決まっている．\(13!\)はint型（端末室の計算機では32 ビット）の最大値を超えるので，この課題では12までにしている．端末室の Linuxの場合，long型も32ビット．64ビットOSなどでは，long型は64ビット．

  • nを13以上にしたければ，int factorial(int n)の代わりに，\(m < n\)を満たす\(m, n\)を引数にとり，\((m+1)\)から\(n\)までの積（\(\frac{n!}{m!}\)）を求める関数を作る方法もある．\(n!\) を求めずに，階乗の途中で\(r!\)などで約分するイメージ．桁あふれしにくくできる．
