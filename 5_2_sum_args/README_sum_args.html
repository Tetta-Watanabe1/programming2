5_2_sum_args

1 説明

コマンドライン引数に与えられた整数の合計を出力せよ．

コマンドライン引数：

引数はそれぞれ整数である．引数の個数はいくつでもよい．

引数が正の整数の場合には，"+10"のように先頭にプラスがつく場合と， "10"のように先頭のプラスを省略する場合がある．負の数の場合， "-10"のように先頭にマイナスがつく．atoi関数で整数に変換できる引数が与えられる．

入力：なし．

出力：コマンドライン引数に与えられた整数の合計．テスト一覧のようにプラス記号は省略すること．

コンパイル：

    $ gcc -Wall -o sum_args sum_args.c

4 ヒント

  • 明快入門C 88 コマンドラインから引数を渡す．Cの絵本 main()関数．

  • main関数の第一引数（int argc）にはコマンド名を含むコマンドライン引数の個数が格納されている．第二引数（char *argv[]）は配列である．配列のそれぞれの要素はコマンドライン引数の文字列を指すポインタである．例えば，ターミナルで次のように実行したとする．

    $ ./sum_args 1 2

    このとき， 次のように第一引数と第二引数に値（やアドレス）が格納されて，main関数が呼び出される．

    argc = 3;
argv[0] = "./sum_args";
argv[1] = "1";
argv[2] = "2";
argv[3] = NULL ;

    コマンドラインの文字列は，1つ以上の空白文字で引数に分割される．

    argv[3]の値NULLはポインタ型の定数である．メモリアドレス0番地を指すポインタで，無効なポインタとして使う．

  • コマンドライン引数に与えられた整数は文字列としてmain関数に渡される．文字列で表現された整数をint型の整数に変換するには，atoi関数を使うとよい．もちろん，atoi関数を使わずに自分で同様の処理を書いても構わない．atoi関数はstdlib.hヘッダファイルに次のように宣言されている．

    #include <stdlib.h>

int atoi(char *str);

    引数のstrは文字列の先頭アドレス（char型配列のアドレス）である． atoi関数の呼び出し例は明快入門C 83 その他の関数にある．このほかに明快入門Cの索引から，48 数値入出力3 数値入力の別方法や付録Aにも atoi関数の説明があることがわかる．

    atoi関数を試すには次のプログラムで"+1"の部分を書き換えるとよい．

    int i;
i = atoi("+1");
printf("%d\n", i);

  • Cygwin64ターミナルで（英文）atoi関数のオンラインマニュアルを読める．マニュアルページを終了するときは Q キーを押す．マニュアルページの使い方を知りたいときは H キーを押す．

    $ man atoi
ATOI(3)              Standard Utility F...

NAME
       atoi, atol, _atoi_r, _atol_r - s...

SYNOPSIS
       #include <stdlib.h>

       int atoi(const char *s);

    man atoi コマンドを実行したときに「atoiというマニュアルはありません」と表示される場合は，Cygwin に cygwin-doc パッケージを追加してから改めてmanコマンドを実行する．パッケージの追加方法はBb9の「必携 PC（Windows 11）の演習準備」ページの「Cygwinのmanコマンドの使い方」という項目をみよ．

    $  setup-x86_64 -P cygwin-doc
