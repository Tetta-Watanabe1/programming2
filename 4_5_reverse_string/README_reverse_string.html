4_5_reverse_string

1 説明

与えられた文字列を反転させるchar *reverse_string(char *p)関数を作成せよ．この関数の戻り値は引数pとすること．main関数は変更しないこと．反転した文字列とは，最後の文字から逆順に並んだ文字列のことである．

reverse_stringは文字列の長さが79文字を超えても正しく動作するようにすること．

入力：標準入力から空白を含まない文字列を読み込む．読み込む文字列長は79文字までとする．

出力：

  • 1行目は入力された文字を反転して出力する．（最後の文字から逆順に出力する）
  • 2行目は入力された文字を2回反転して出力する．（元の文字列）

reverse_string関数：

  • 文字列へのポインタchar *pを引数にとる．
  • 戻り値は引数p．関数を入れ子にして呼び出せるようにするため．
  • 引数の文字列を反転させる．

コンパイル：

    $ gcc -Wall -o reverse_string reverse_string.c

4 ヒント

  • 明快入門C 53 関数の構成（関数の方と引数），67 ポインタによる配列渡し．

  • main関数を読んで理解しよう．入力された文字列はmain関数の配列 sに格納され，s2にもコピーされる．

    reverse_string(s)を呼び出し，reverse_stringによって反転された文字列sを出力する．

    tには文字列s2を2度反転した文字列が返る．受け取ったtを出力する． 2度反転すると元に戻るが，最初の文字列を出力しているわけではない．

  • 次の2文でtには文字列s2を2度反転した文字列が入る．

    t1 = reverse_string(s2);
t = reverse_string(t1);

    この2文は次の1文でまとめて書ける．この1文でもtに同じ文字列が入る．内側のreverse_string(s2)の戻り値は，そのまま外側のreverse_string関数の引数として渡される．

    t = reverse_string(reverse_string(s2));

  • main関数のs, s1, t，t1がよくわからない場合は次のprintf文を確認したい場所に入れると，これらのアドレスを出力できる．return文を適切に書けば，s1と同じアドレスがtとt1に格納される．

    printf("%p %p %p %p\n", s, s2, t, t1);

  • reverse_string関数は文字列へのポインタを引数にとる．戻り値は引数p なので，次のように定義する．この例では仮引数名をpにしている．

    char reverse_string(char *p) {
  // 反転処理
  return p; // このpは反転処理の間も変更しない．
}

  • 引数のポインタpは参照渡しなので，pの指す配列要素（p[0]や*p）を読み書きすると，main関数の配列sの要素を読み書きすることになる．（明快入門C 67 ポインタによる配列渡し）

  • 文字列を反転するには，先頭の文字と末尾の文字を入れ替え，先頭の次の文字と末尾の1文字前を入れ替えるという処理を，文字列の中間まで繰り返す．以降，この方法で反転する手順を説明する．

  • 仮引数pの代わりに配列s[]を用いる例を示す．この例では，文字列の長さを覚える変数を使わずに末尾の文字までポインタを移動する．

    char s[] = "This_is_a_pen.";
char *head; // 文字列の先頭から後ろに動かすポインタ
char *tail; // 文字列の末尾から前に動かすポインタ
head = s;
tail = s;

    この例ではheadとtailはs[0]を指すポインタになる．代入文などで s[0]と書いても*headや*tailと書いても同じ配列要素になる． head++;するごとにs[1]，s[2]，s[3]，\(\ldots\)とポインタが指す文字が移動する．なお，head = s;はhead = &s[0];と書くこともできる．

  • 文字列s[]の終端にはヌル文字'\0'が格納されているので，先ほどの例に続けて次のように書くと，ポインタtailはヌル文字まで移動した後，1文字戻る．つまり，ポインタtailは文字列の末尾の文字（この例ではピリオド）が格納されている配列要素を指すようになる．

    while (*tail != '\0')
  tail++;
tail--;

  • ここまでに説明した方法で，文字列の先頭文字を指すポインタheadと文字列の末尾のヌル文字の前の文字を指すポインタtailが用意できる．

  • *headと*tailの文字を入れ替えるには，別の変数（例えばtmp）にどちらかの値を覚えておいて，入れ替える．つまり，次のようにする．

    char tmp;
tmp = *head;
*head = *tail;
*tail = tmp;

    次のように書くと，*headの文字が失われる．

    *head = *tail;
*tail = *head;

  • headとtailが常に先頭からi文字目と末尾からi文字目を指すように移動するには，二つのポインタを次のように1文字ずつ前や後ろにずらす．

    head++;
tail--;

  • ポインタheadとtailの大小関係を比較することで，二つのポインタが指している配列要素の添字を比較できる．headがs[i]を指し，tailが s[j]を指しているとすると，head < tailならばi < jである．head < tailを満たす間，ポインタをずらしながら，文字を入れ替え続けると，文字列の中間まで繰り返すことができる．

    printf("%p %p\n", head, tail);
while (head < tail) { // 文字列の中間まで
  head++;
  tail--;
}
printf("%p %p\n", head, tail);

  • 文字列の長さを調べるためにstrlen関数を使ってもよい．strlen関数は string.hヘッダファイルで宣言されている．while (*tail != '\0')の部分をstrlen関数を使って書き換えることができる．
