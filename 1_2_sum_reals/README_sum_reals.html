1_2_sum_reals

1 説明

n個の実数の和を出力せよ．

入力：

  • 標準入力の最初は実数の個数n． nは1以上．テストではnは50以下．
  • それに続いてn個の実数．

出力：

  • n個の実数の和を小数第1位まで表示する．

注意：

  • 出力の行末は改行すること．今後の課題についても同様に最終行末は改行すること．

  • 和を求める変数は0.0を代入して初期化すること．下に例を示す．

      float sum; // または double sum;
  sum = 0.0;  // これを忘れがち
  for (...) {
    sum += ...; // sum = sum + ...; 右辺の sum の値は最初は0.0
  }

コンパイル：

    $ gcc -Wall -o sum_reals sum_reals.c

4 ヒント

  • 総和を求めるときに，入力されるn個の実数を一旦配列に格納したくなる．しかし，すべての実数を読み込んでから合計を計算するのではなく，読み込むたびに合計を更新すれば配列を使う必要はない．

  • 入力nに50より大きい整数が入力された時の動作は問わない．

  • 標準入力から実数を読み取るにはfloat型変数とscanf関数を使うとよい．

    float a;
scanf("%f", &a);

    float型変数の代わりにdouble型変数に読み込む場合はscanf関数の書式指定に"%lf"を使う．ただし，テストではfloat型の変数を使うことを想定している．
