5_3_sort_args

1 説明

コマンドライン引数で与えられる三つまたは四つの単語を辞書順に出力せよ．

コマンドライン引数：三つ，または四つの単語

入力：なし

出力：単語（コマンドライン引数）が三つの場合

  • 辞書順で最も小さい単語（辞書で最初に出てくる単語）
  • 辞書順で二番目に小さい単語
  • 辞書順で最も大きい単語（辞書で最後に出てくる単語）

単語が四つ場合

  • 辞書順で最も小さい単語
  • 辞書順で二番目に小さい単語
  • 辞書順で三番目に小さい単語
  • 辞書順で最も大きい単語

コンパイル：

    $ gcc -Wall -o sort_args sort_args.c

4 ヒント

  • 辞書順は apple，banana，cherry のように単語が辞書に載っている順番． banana，apple，cherry は banana と apple が逆順なので辞書順ではない．

  • 辞書順に並んでいることを確認するには，すべての隣り合う二つの間で辞書順になっていることを確認すればよい．確認回数は三単語なら二回，四単語なら三回．単語数が\(n\)なら\(n-1\)回．

  • 文字列比較関数は文字列処理ですでに学んでいます．strcmp関数．明快入門C 付録A C言語主要関数一覧 文字列処理 が調べやすい．

  • まずは 文字列比較関数の使い方を思い出すために，三単語が辞書順に並んでいること確認するプログラムを書くのがいいでしょう．単語は自分で入力しなさい．なお，最初のテストは三単語で，辞書順に並んでいます．

    その後，三単語の場合のプログラムを作りましょう．

    最後に，四単語の場合もプログラムにしましょう．

  • 三つのコマンドライン引数の場合，すべての場合（順列）を並べると，\(3! = 6\) 通り．a b c, a c b, b a c, b c a, c a b, c b a

  • 四つのコマンドライン引数がある場合は，すべての場合を並べると，\(4! = 24\) 通り．すべての場合をプログラム中に並べるのは，闇雲に並べると，いくつかの場合を漏らしそうです．

  • 仮に四つのコマンドライン引数を単語 \(a, b, c, d\)．先頭から一単語ずつ辞書順に揃えていく方法があります．この方法を挿入ソートと呼びます．ただし，辞書順で\(a<b<c<d\)とは限らないことに注意せよ．

    1. まず単語\(a\)だけ考える．辞書順に並んでいるといえる．リストを\((a)\) とする．
    2. 次に辞書順を維持したまま，リスト\((a)\)に単語\(b\)を加える．加えたリストを\((a', b')\)とする．\(a' < b'\)を満たす．\(a'\)と\(b'\)は一方が\(a\) で他方が\(b\)．
    3. さらに辞書順を維持したまま，リスト\((a', b')\)に単語\(c\)を加える．加えたリストを\((a'', b'', c'')\)とする．\(a'' < b'' < c''\)を満たす．
    4. 最後に辞書順を維持したまま，リスト\((a'', b'', c'')\)に単語\(d\)を加える．加えたリストを\((a''', b''', c''', d''')\)とする．\(a''' < b''' < c''' < d'''\)を満たす．
    5. リスト\((a'', b''', c''', d''')\)は辞書順に並んでいる．

  • 「辞書順を維持したまま，リスト\((a', b')\)に単語\(c\)を加える」部分をもう少し詳しくみる．\(c\)より小さいものが見つかるまで，リスト末尾から先頭に向かって一単語ずつ\(c\)と比べる．\(c\)より小さいものが見つかったらそのすぐ後に\(c\)を入れて終わる．

    適切な場所を探すイメージ(b?は単語bが入る仮の位置)

    1. リスト (a', b') はすでに辞書順．
2. リスト (a', b', c?) を辞書順と仮定する．
3. リスト (a', b', c?) について b' <= c ならば，このリスト (a', b', c) は辞書順．
4. そうでなければ（つまり c < b' ならば），リスト (a', c?, b') を辞書順と仮定する．
5. リスト (a', c?, b') について a' <= c ならば，このリスト (a', c, b') は辞書順．
6. そうでなければ（つまり c < a' ならば），リスト (c?, a', b') を辞書順と仮定する．
7. リストの先頭と比べたので繰り返しは終わり．リスト (c, a', b') は辞書順．

    上の手順をC言語っぽく書くと次のようになる．単語c <= 単語tなどの比較は単語の比較をしているので文字列比較関数を使う．C言語で c <= t と書くとアドレスの比較になる．

    単語t = 単語c;
if (単語b <= 単語t) {   // 3
  単語c = 単語t;        // 3
} else {               // 4
  単語c = 単語b;        // 4
  if (単語a <= 単語t) { // 5
    単語b = 単語t;      // 5
  } else {             // 6
    単語b = 単語a:      // 6
    単語a = 単語t;      // 7
  }
}

  • 四つ目の単語dを加えるところも，三つ目の単語の場合と同様です．最後の単語cから順に比較して適切な場所に入れます．
