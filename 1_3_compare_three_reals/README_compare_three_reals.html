1_3_compare_three_reals

1 説明

標準入力の3つの実数が大きい順に並んでいるならYesを，大きい順でなければNoを出力せよ．

入力：

  • 標準入力から3つの実数をfloat型変数に読み込む．

  • 読み込んだ実数を先頭から順に\(a, b, c\) とする．\(a, b, c\) は異なる値と仮定してよい．

出力： \(a > b > c\)が成立すればYesを，成立しなければNoを出力する． \(a > b > c\)は，\(a > b\)かつ\(b > c\)であることに注意せよ．また，行末は改行すること．

コンパイル：

    $ gcc -Wall -o compare_three_reals compare_three_reals.c

4 ヒント

  • 標準入力からfloat型の変数に実数を読み込むときは書式に"%f"を指定する．

  • 3つの変数aとbとcが大きい順に並んでいることを，次のように一つの式で表すことはできない．演算子の優先順位の結合規則を理解すると納得できる．

    if (a > b > c)

    そのため，a > bかつb > cのように分解して，「かつ」の部分を論理積（&&）で表して次のように書く．

    if ((a > b) && (b > c))

    演算子の優先順位は>の方が&&より高いので，次のようにかっこを削除できる．

    if (a > b && b > c)

  • 論理積演算子（&&）は二つのif文に分解して同じ場合分けにできる．

    if (a > b && b > c)
  // a > b かつ b > c のときに実行したい文
else
  // (a > b かつ b > c)ではないときに実行したい文
  // 言い換えると
  // a > b ではないとき，あるいは b > cではないときに実行したい文
  // a <= b または b <= c のときに実行したい文

    上の一つのif文を二つのif文に分解して次のように書ける．ただし，「a > b > c ではないときに実行したい文」を2箇所に同じように書くことになるので，論理積演算子（&&）を使ったほうがすっきり書ける．

    if (a > b) {
  if (b > c)
    // a > b > c のときに実行したい文
  else 
    // a > b > c ではないときに実行したい文
} else {
  // a > b > c ではないときに実行したい文

  • if (a > b > c)の条件式は文法的には正しいのでコンパイルエラーにはならなず，正常にコンパイルが終了する．ところがif (a > b && b > c)とは違う動作をする．なぜなら，>演算子が左結合であることから， (a > b) > cと解釈されるからである．ここで，式a > bの値が0または1 であることに注意すると，(a > b) > cは次の4つに場合分けできる．

    1. aがbより大きく，cが1より小さい．
    2. aがbより大きく，cが1以上．
    3. aがb以下で，cが0より小さい．
    4. aがb以下で，cが0以上．

    式a > b > cの真偽を場合ごとに表に示す．表には本来調べたかったa, b, cが大きい順に並んでいる例や並んでいない例を場合ごとに示す．

    場合 条件式a > b > cの真偽 a, b, cが大きい順に並んでいる例 大きい順ではない例
    (i) 真 a = 4.0; b = 3.0; c = 0.0; a = -2.0; b = -3.0; c = 0.0;
    (ii) 偽 a = 4.0; b = 3.0; c = 2.0; a = -2.0; b = -3.0; c = 2.0;
    (iii) 真 なし a = 3.0; b = 4.0; c = -1.0;
    (iv) 偽 なし a = 3.0; b = 4.0; c = 1.0;
