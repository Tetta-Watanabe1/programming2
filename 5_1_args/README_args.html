プログラミングII演習課題の手引き
5_1_args

1 説明

コマンドライン引数の数（コマンド名自身を含む数）とそれぞれの引数を出力するプログラムを作成せよ．

コマンドライン引数：なし，または1個以上．

入力：なし．

出力：

  • 1行目：コマンドライン引数の数（コマンド名自身を含む数）

  • 2行目以降：実行例のようにコマンドライン引数の番号と引数を1行に一つずつ．コマンド名自身は含めない．

コンパイル：

    $ gcc -Wall -o args args.c

2 テスト一覧

テスト番号: 1, 2, 3, 4

（1行が長い場合にブラウザでは複数行に分けて表示される場合あるので注意すること）

4 ヒント

  • 明快入門C 88 コマンドラインから引数を渡す．Cの絵本 main()関数．

  • コマンドラインに指定された引数を読み取るプログラムはmain関数の引数を二つ使う．第1引数のargcはコマンドライン引数の個数（コマンド名自身を含む数）である1．コマンド名も数に含めるので，実行するコマンドラインがテスト1のように ./args first_argument second_argument third_argument であれば個数は4つである．第2引数のargvは配列で，配列のそれぞれの要素が文字列のアドレスである．

    int main(int argc, char *argv[])

    例えば，argv[0]はchar *型，つまりchar型ポインタである． argv[0]にはコマンド名の文字列の先頭アドレスが格納されている（実行例では"./args"の先頭アドレス）．*argv[0]とargv[0][0]はどちらも同じ文字型（char型）の変数である．この変数の値は，コマンド名の先頭文字である（実行例ではこの文字はピリオド’.’）．

  • 実行例をよく見なさい．最初に出力する引数の数にはコマンド名を含みますが，続く出力にはコマンド名（argv[0]）は含みません．

  • argc番目のコマンドライン引数（argv[argc]）にNULLが格納されていることを確認しなさい．

    if (argv[argc] == NULL) {
  printf("argv[%d] is NULL\n", argc);
} else {
  printf("argv[%d] is NOT NULL\n", argc);
}

  • argcとargvという仮引数名が一般的に使われる．他の名前でもエラーになるわけではないが，他の名前に変える理由もないのでこのまま覚えるといいだろう．argcはargument countの略で，argvはargument vectorの略と覚えるとよい．

  1. コマンドライン引数は空白で区切られる．↩︎
