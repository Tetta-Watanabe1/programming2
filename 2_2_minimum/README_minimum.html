プログラミングII演習課題の手引き
2_2_minimum

1 説明

二つの整数を引数に取り，そのうち小さい方を返すint min(int x, int y)関数を書け．main関数は書き換えないこと．

min関数の引数：

  • 第1引数: int型の整数．仮にxと呼ぶ．
  • 第2引数: int型の整数．仮にyと呼ぶ．

min関数の戻り値：

  • xとyのうち小さい方の整数（int型）．
  • xとyが同じ場合はその値（xまたはy）．

入力：標準入力に3つの整数が与えられる．

出力： 3つの整数のうち最も小さい整数を表示する．

コンパイル：

    $ gcc -Wall -o minimum minimum.c

#include <stdio.h>

// ここにint min(int a, int b)関数を書く

int main() {
  int a, b, c, m;

  scanf("%d", &a);
  scanf("%d", &b);
  scanf("%d", &c);

  // aとbのうち小さいものをmに代入
  m = min(a, b);

  // 三つのうち最も小さいものをmに代入
  m = min(m, c);

  printf("%d\n", m);
  return 0;
}
