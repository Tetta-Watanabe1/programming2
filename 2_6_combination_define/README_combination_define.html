プログラミングII演習課題の手引き
2_6_combination_define

1 説明

発展課題

条件付きコンパイルを使う課題である．作成するプログラムはVERBOSEを定義せずにコンパイルした場合は単純な出力になり，定義してコンパイルした場合は冗長な出力になるプログラムとせよ．

先ほどの課題 combination.c と同様に組合せの数 \(_n\mathrm{C}_r = \frac{n!}{r!(n-r)!}\) を出力するプログラムとする．ただし，コンパイル時にVERBOSEが定義されている場合は，\(n=4, r=1\)のとき24 / 1 / 6 = 4のように，計算式を出力すること．

入力：

  • 標準入力に2つの整数\(n\)と\(r\)が，この順に与えられる．

  • \(n\)と\(r\)はいずれも整数で，\(1\le n < 12\)，\(1\le r \le n\)とする．

出力：

  • （VERBOSEを定義しないときの単純な出力）

    組合せの数\(_n\mathrm{C}_r\)を出力する．

  • （VERBOSEを定義するときの冗長な出力）

    4つの整数\(n!\)，\(r!\), \((n-r)!\), \(_n\mathrm{C}_r\)を %d / %d / %d = %d の形式で出力する．

コンパイル： VERBOSEを定義しない場合と定義する場合の二つの実行ファイルを作れ．

（VERBOSEなし）

    $ gcc -o combination combination.c

（VERBOSEあり）

    $ gcc -o combination_verbose -DVERBOSE combination.c

#include <stdio.h>

/* 先ほどと同様に，ここに作成した int factorial(int n) 関数を定義する */

int main() {
  int n, r;

  scanf("%d", &n);
  scanf("%d", &r);
  printf("%d\n", factorial(n) / factorial(r) / factorial(n - r));
  return 0;
}

4 ヒント

  • ソースファイル内で#ifdef VERBOSEの行と#endifの行で囲んだ部分を作る．囲まれた行は-DVERBOSEを付けてコンパイルしたときだけコンパイルされる．-DVERBOSEを付けずにコンパイルしたときは，囲まれた行はすべて無視される．

    int main() {
  printf("see no evil, ");
#ifdef VERBOSE
  printf("hear no evil, ");
#endif // VERBOSE
  printf("speak no evil.\n");
}

  • コンパイル時にVERBOSEを定義するには，次のように-DVERBOSEオプションを追加する．

    $ gcc -DVERBOSE combination.c

    ソースファイル中に#define VERBOSEと書くことでも定義できるが，この課題ではコンパイル時に定義と未定義を切り替えるので#defineは使わない．

  • コンパイル時の -o オプションの後のcombinationと combination_verboseは実行ファイル名．

  • 二つのgccコマンドをまとめて実行したいときはセミコロンで区切って並べるとよい．

      $ gcc -o combination combination.c; gcc -o combination_verbose -DVERBOSE combination.c

余談：

  • 「2進数表現」の先取り： int型やlong型は整数を表すが，それぞれ表せる数値の最大値が決まっている．\(13!\)はint型（端末室の計算機では32 ビット）の最大値を超えるので，この課題では12までにしている．端末室の Linuxの場合，long型も32ビット．64ビットOSなどでは，long型は64ビット．

  • nを13以上にしたければ，int factorial(int n)の代わりに，\(m < n\)を満たす\(m, n\)を引数にとり，\((m+1)\)から\(n\)までの積（\(\frac{n!}{m!}\)）を求める関数を作る方法もある．\(n!\) を求めずに，階乗の途中で\(r!\)などで約分するイメージ．桁あふれしにくくできる．

  • #endifがどの#ifdefに対応するか分かりやすくするために，次のようにコメントを書く習慣がある．

    #ifdef VERBOSE
#endif // VERBOSE
