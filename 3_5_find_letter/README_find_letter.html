3_5_find_letter

1 説明

int find_letter(char str[], char x)関数を作成せよ．この関数はstrの先頭から何文字目に文字xがあるかを返す．文字xがない場合は-1を返す．文字列str[]の中に文字xが複数ある場合は，最初の位置を返す．文字列 strと文字xはいずれもfind_letter関数の引数であることに注意せよ．

さらに，main関数にこの関数を呼び出す一文を追加せよ．

find_letter関数の引数：

  • 第1引数：文字列が格納されたchar型配列str[]
  • 第2引数：char型変数x
  • 配列strに格納されている文字列の長さは100文字以下とは限らない．

find_letter関数の戻り値：

  • 配列 str[]に格納された文字列のi文字目（str[i]）が文字xならiを返す．
  • ただし，配列 str[]中に文字xが複数ある場合は先頭に近い方のiを返す．
  • 配列 str[]中に文字xが現れない場合は-1を返す．

入力：

  • 標準入力から探す文字を変数xに読み込む．

  • 続いて，空白を含まない文字列を配列strに読み込む．長さが100文字を超える場合は100文字まで読み込む．find_letter関数はstrの長さを制限する必要はない．

出力：

  • 文字列str中の文字xの位置を表示する．文字列の最初の文字がxであれば0を，2番目の文字がxであれば1を，\(i\)番目の文字がxであれば \(i-1\)を出力する．

  • 文字xが文字列str中に存在しない場合には-1を出力する．

  • 文字列中に2回以上，xが出現する場合には，最初の位置を出力する．

コンパイル：

    $ gcc -o find_letter find_letter.c

4 ヒント

  • char型配列str[4]に文字列"ABC"が格納されているとき，最後の文字 'C'の後に'\0'（ヌル文字と呼ぶ1文字）が格納されている．例えば，次のように配列str[]を初期化すると，strの要素数は4で，str[3]には '\0'が格納される．

    char str[] = "ABC";

    これは，次のように配列を初期化する場合と同じ．

    char str[4];
str[0] = 'A';
str[1] = 'B';
str[2] = 'C';
str[3] = '\0';

  • 文字列str[]の先頭文字から順にたどるには次のようにfor文が使える．末尾に'\0'があるので，それが見つかるまでforループの本体を繰り返す．

    int i;
for (i = 0; str[i] != '\0'; i++) {
  printf("%c", str[i]); // 試しにi番目の文字を出力する
}

    forの代わりにwhileを使っても書ける．

    int i;
i = 0;
while (str[i] != '\0') {
  printf("%c", str[i]); // 試しにi番目の文字を出力する
  i++;
}
