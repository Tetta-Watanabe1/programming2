プログラミングII演習課題の手引き
3_4_mytoupper

1 説明

char lower2upper(char c)関数を作成せよ．この関数は，引数の文字cが英小文字の場合に英大文字に変換して返し．それ以外の場合は引数の文字をそのまま返す．ただし，lower2upper関数内で標準ライブラリctype.hの toupper関数とislower関数を使ってはならない．

さらに，main関数にこの関数を呼び出す一文を追加せよ．

このプログラムは，標準入力から入力した1行分の文字列を出力する．ただし，入力された文字のうち英小文字は英大文字に変換されて出力される．このようなプログラムになるようにせよ．

lower2upper関数の引数：文字c（char型）

lower2upper関数の戻り値：文字cが英小文字ならcを大文字にして返す．それ以外はcを返す．

入力： 1行分の文字列を読み込む．行は改行文字\nで終わる．行には空白や記号を含みうる．

出力：読み込んだ文字列を出力するが，アルファベット小文字だけを大文字に変換し，その他の文字は変換せずに出力する．

コンパイル：

    $ gcc -o mytoupper mytoupper.c

#include <stdio.h>

/* lower2upper関数をこの辺りに定義する */

int main() {
  char c;
  while ((c = getchar()) != '\n') {

    /* ここでlower2upper関数を呼び出す．引数は変数c．戻り値はcに代入する */

    printf("%c", c);
  }

  return 0;

4 ヒント

  • char型変数cに格納された文字は整数値として扱えるため，大小関係を比較できる．それぞれの文字に割り当てられた整数はASCIIコード表で確認できるが，この整数を覚える必要はない．シングルクオートで囲んだ文字定数で書く．例えばchar型変数cは以下のように他の文字'b'などとの大小比較や加減算ができる．

    char c;
c = 'a';
if (c < 'b') // c < 98 とも書けるが書かない
  printf("c < 'b'\n");
if (c == 'c')
  printf("c == 'c'\n"); // c == 99 とは書かない
c += 'A' - 'a';
printf("%c\n", c); // 文字'A'が出力される

  • 英小文字はASCIIコード表で'a'，'b'，'c', \(\ldots\)，'y'， 'z'のように隙間なく並んでいる．よって，文字cが英小文字（lowercase letter）か調べるには次のようなif文でよいだろう．

    if (c >= 'a' && c <= 'z')
  // cが英小文字である場合の処理
  printf("'%c' is a lowercase letter\n", c);
else
  // cが英小文字でない場合の処理
  printf("'%c' is NOT a lowercase letter\n", c);

    次のように<=演算子を使うように順序を入れ替えて書くほうが文字cが 'a'から'z'の間ということがわかりやすいかもしれない．

    if ('a' <= c && c <= 'z')

    このif文の条件を関数にしたものがctype.hで宣言されている islower関数．ただし，冒頭で説明したように，この課題ではislower関数とtoupper関数を使ってはならない．islower関数については明快入門 C p.198やman islowerなどを参照．

    #include <ctype.h>

if (islower(c))

-　作成するlower2upper関数は標準ライブラリctype.hのtoupper関数と同じように文字変換をする．

  • lower2upper関数でgetcharやscanf，printfなどの入出力関数を呼び出す必要はない．
